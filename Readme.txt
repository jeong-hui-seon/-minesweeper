<설치 및 실행 방법>
1. OpenFrameworks 파일의 Project Generator를 사용하여 지뢰찾기 프로젝트를 생성한다. 이때 프로젝트는 OpenFrameworks - apps - myApps 디렉토리에 위치시킨다.
2. 제공된 지뢰찾기 프로그램의 소스 파일들을 프로젝트의 src 폴더의 기존 파일들에 덮어씌운다.
3. 프로젝트를 빌드하고 실행한다.

<기능>
1. New Game : 새로운 게임을 시작한다. 초기화면에서 게임을 시작할 때마다 새로운 지뢰 배치와 게임 보드가 설정된다. 이는 프로젝트 실행 시 자동으로 시작된다.
2. Flag : 마우스 우클릭을 통해 지뢰가 의심되는 위치에 깃발을 표시하거나 제거한다.
3. Reveal : 마우스 좌클릭으로 타일을 선택하고 열어 지뢰의 유무를 확인한다. 지뢰가 있는 타일을 클릭하면 게임이 종료된다.
4. Exit : 게임 오버 또는 승리 후 'n' 키를 눌러 프로그램을 종료할 수 있다.
5. Restart : 게임 오버 또는 승리 후 'y' 키를 눌러 게임을 다시 시작할 수 있다. 
6. Show Remaining Flags : 화면의 상단에 남은 깃발의 수를 실시간으로 표시한다.
